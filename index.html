The Keytab Vault: Diskless Bare-Metal Linux with AD Auth via SSSD

July 2026 · infra · linux · active-directory

A PXE RAM-boot architecture for bare-metal Linux fleets that stream a fresh OS into RAM on every boot and pull an AD Kerberos keytab from a secrets vault at boot time — so the disk stays stateless and the AD computer object is created exactly once, ever.

Built with plain HTML and CSS, hosted on GitHub Pages.
